# TransitSim4 Toy Optimizer

Open `index.html` in a local browser to run the simulation.

## What it does
- Random journeys across a rectangular world.
- Evaluates travel time by walking vs using train lines with stations.
- Uses acceleration/deceleration for train segments between stations.
- Adds boarding and transfer penalties.
- Optimizes line geometry, station placement, and line count within bounds.
- Runs multiple worker threads for faster search.
